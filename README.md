<h3>多列展示且后面出现展开收起</h3>
<b>缺点一大堆，先列举下其缺点:</b>
<ul>
    <li>只支持chrome和手机端浏览器</li>
    <li>好几个地方都使用像素计算高度和宽度,使用的时候需要自己调试到符合自己项目的高宽</li>
    <li>展开按钮(当然也可以是其他的东西,自己写)一定是在期望的最后一列右侧,即使文字只有一行它也会这样</li>
    <li>当文字长度不够的时候,省略号依然会显示出来</li>
    <li>因为它有如此多的缺点,我就不写成插件了(主要是因为上面第二条)</li>
</ul>
